# 🧠 JS Think — Aprende a pensar en JavaScript

Una app interactiva para aprender JavaScript desde cero, enfocada en desarrollar el pensamiento lógico de programación.

## ✨ Características

- 12 desafíos progresivos organizados en 6 niveles
- Editor de código con consola integrada
- Feedback inteligente por desafío: te guía sin darte la respuesta directa
- Seguimiento de progreso visual
- Pistas y ejemplos disponibles cuando los necesitas
- **Sin dependencias. Sin instalación. Un solo archivo HTML.**

## 📚 Contenido

| Nivel                | Temas                                             |
| -------------------- | ------------------------------------------------- |
| 📦 Variables & Tipos | let, const, string, number, boolean, null, typeof |
| 🔀 Condicionales     | if, else, else if, operadores de comparación      |
| 🔁 Loops             | for, operador módulo %                            |
| ⚙️ Funciones         | function, parámetros, return, arrow functions     |
| 📋 Arrays            | índices, length, .map(), .filter()                |
| 🗂️ Objetos           | propiedades, acceso con punto, arrays de objetos  |

## 🚀 Cómo usar

1. Descarga o clona este repositorio
2. Abre `index.html` en tu navegador (Chrome recomendado)
3. ¡Listo! No necesitas instalar nada

```bash
git clone https://github.com/tu-usuario/js-think.git
cd js-think
# Abre index.html en tu navegador
```
